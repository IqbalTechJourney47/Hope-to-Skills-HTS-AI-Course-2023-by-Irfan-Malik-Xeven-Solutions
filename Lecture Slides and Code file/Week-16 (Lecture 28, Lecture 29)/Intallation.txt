Check PyArrow Installation:
Ensure that the pyarrow library is installed in your "bot" Conda environment. You can check this by running the following command:

conda list pyarrow
----------------------------------------

If it's not installed, you can install it using:

conda install -c conda-forge pyarrow
----------------------------------------

Update PyArrow:
If pyarrow is already installed, you might want to update it to the latest version to resolve potential compatibility issues:

conda update -n bot pyarrow
----------------------------------------

conda install --upgrade pyarrow
conda install --update pyarrow
conda install --update-deps pyarrow
